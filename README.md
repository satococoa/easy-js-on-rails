# やさしい AJAX on Rails サンプルコード

## スライド

https://speakerdeck.com/satococoa/easy-ajax-on-rails

## 動かし方

```shell
$ bin/setup
$ bin/rails server
```

## ブランチ名とスライド中の呼び名の対応関係

- master: 非 AJAX
- js-erb-pattern: \*.js.erb
- partial-on-json-pattern: Partial on JSON
- object-on-json-pattern: Object on JSON
- omake: おまけ
